# Class Event #

### Method addEvent ###

Adds a new event.

> - Input Parameters
    * actionId : int
    * startTime : datetime
    * endTime : datetime

> - Output Parameter
    * returns event object

### Method editEvent ###

updates an event.

> - Input Parameters
    * event : Event

> - Output Parameter
    * returns event object

### Method deleteEvent ###

deletes an event.

> - Input Parameters
    * eventId : int

> - Output Parameter
    * returns boolean

### Method searchEvent ###

searches an event.

> - Input Parameters
    * searchText : string

> - Output Parameter
    * returns event object
