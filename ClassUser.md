# Class User #

### Method Register User ###

Creates a new user.

> - Input Parameters
    * userBasicInfo
    * userCommInfo
    * userExtraInfo

> - Output Parameter
    * returns user object

### Method Login User ###

The user log into the system with an e-mail address and a password.

> - Input Parameters
    * e-mail address
    * password

> - Output Parameter
    * none

### Method Get Users ###

The method gets the list of all users.

> - Input Parameters
    * none

> - Output Parameter
    * list of all users
