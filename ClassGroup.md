# Class Group #

### Method Add Group ###

Creates a new group.

> - Input Parameters
    * new group's data

> - Output Parameter
    * returns group object

### Method Edit Group ###

Edits allowed group parameters.

> - Input Parameters
    * Group Object

> - Output Parameter
    * none

### Method Search Group ###

The method gets the list of all groups.

> - Input Parameters
    * Searching name

> - Output Parameter
    * list of all groups

### Method Delete Group ###

The method deletes the existing group.

> - Input Parameters
    * Group object

> - Output Parameter
    * none
