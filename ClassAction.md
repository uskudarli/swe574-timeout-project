# Class Action #

### Method Add Action ###

Creates a new action.

> - Input Parameters
    * new action's data

> - Output Parameter
    * returns action object

### Method Edit Action ###

Edits allowed action parameters.

> - Input Parameters
    * Group Action

> - Output Parameter
    * none

### Method Delete Action ###

The method deletes the existing action.

> - Input Parameters
    * Action object

> - Output Parameter
    * none
